The Rapid Framework Project
===========================

The Rapid Framework project is a project aimed an making Continuous Integration
Pipelines, a process that is customizable and friendly to DevOps Organizations.

There are many utilities out there, but Rapid seeks to be a very lightweight
solution that simple, fast, and still powerful to do all you need to in your
delivery process.

Parts of the Rapid Framework
============================

Server - The Master server that records and orchestrates test runs.

Client - The Client server that executes and runs the needed tests.

Results Engine - The Engine that parses and records the results of tests.

Test Analysis Engine - An Engine used to analyze the test suite and suport
the Behavior Point philosophy.
